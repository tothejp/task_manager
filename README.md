# 임무분담표 (TaskShare)

소규모 팀(5~15명)을 위한 역할 분리형 임무 분담 관리 앱입니다. 팀원은 모바일로 본인의 일정과 휴가를 입력하고, 관리자는 PC에서 이를 종합해 가용 인원을 파악하고 과업을 배정합니다.

## 왜 만들었나

엑셀이나 종이로 업무 분담표를 관리하면 휴가/부재가 생길 때마다 수동으로 갱신해야 하고, 특정 인원에게 업무가 몰리는 문제를 알아차리기 어렵습니다. TaskShare는 이 과정을 자동화하고 시각화합니다.

## 누구를 위한 앱인가

- 5~15명 규모의 팀
- 사내 부서, 매장, 식당, 소규모 사무실 등 정기적으로 임무/당직을 분배해야 하는 조직

## 주요 기능

### 팀원 (모바일)
- **개인 일정 입력**: 휴가, 휴무일(반복 가능), 근무 가능 시간대를 캘린더에서 등록
- **즉시 반영**: 별도 승인 없이 등록 즉시 관리자 화면에 반영
- **배정 확인 & 완료 체크**: 본인에게 배정된 과업을 확인하고 완료 처리

### 관리자 (PC)
- **가용인원 대시보드**: 날짜를 선택하면 전체 구성원을 가용/휴가/휴무로 자동 분류해서 보여줍니다.
- **스킬 부여 및 필터링**: 구성원에게 자격/스킬 태그를 부여하고, 특정 스킬 보유자만 가용인원 중에서 필터링해 조회할 수 있습니다.
- **과업 관리**: 과업 이름, 진행 시간, 요구 인원수, 필수 스킬을 지정해 생성합니다.
- **Drag & Drop 배정**: 가용 인원 카드를 과업 슬롯으로 드래그해서 배정합니다.
  - 필수 스킬이 없는 인원을 배정하려 하면 경고 모달이 뜨고, 확인을 누르면 배정이 진행됩니다 (강제 차단 아님).
  - 같은 시간대에 이미 다른 과업에 배정된 인원은 배정할 수 없습니다 (시간이 겹치지 않으면 같은 날 여러 과업 배정 가능).
- **자동배정**: 버튼 한 번으로 미배정 슬롯에 스킬·가용성·업무량을 고려한 추천 배정을 일괄 적용합니다. 인원이 부족하면 사유와 함께 미충원으로 표시됩니다.
- **휴가 발생 시 재배정**: 이미 배정된 과업 기간에 휴가가 새로 등록되면 해당 배정이 공백으로 표시되고, 같은 자동배정 로직으로 재배정 후보를 제시합니다.
- **공정성 지표**: 구성원별 누적 배정량을 그래프로 보여주고, 평균보다 ±20% 이상 차이가 나면 경고를 표시합니다.

## 배정 정책 요약

| 상황 | 처리 방식 |
|---|---|
| 필수 스킬 미보유자 배정 시도 | 경고 모달 → 관리자 확인 시 진행 가능 |
| 동일 인원, 시간 겹치는 과업 중복 배정 시도 | 차단 (시간이 안 겹치면 허용) |
| 휴가 등록 즉시 반영 여부 | 승인 없이 즉시 반영 |
| 관리자 모바일 접속 | 조회만 가능 (배정/생성은 PC에서만) |

## 기술 스택

- **Frontend**: Next.js 14, TypeScript, Tailwind CSS, Zustand, React Query
- **Drag & Drop**: dnd-kit
- **Backend/DB**: Supabase (PostgreSQL, RLS, Auth)
- **ORM**: Prisma
- **배포**: Vercel

AI 추천(스킬 매칭, 자동배정 정렬)은 모두 규칙 기반 로직으로 구현되며 별도의 LLM API 호출이 필요하지 않습니다.

## 현재 범위에 없는 기능 (다음 단계)

- 외부 캘린더/메신저(카카오·슬랙) 알림 연동
- 스킬 숙련도 등급(상/중/하)
- LLM 기반 추천 사유 설명, 텍스트에서 스킬 자동 추출
- 멀티팀 지원, 임무 템플릿 저장

자세한 기능 정의는 `task_manager_PRD.md`를 참고하세요.
